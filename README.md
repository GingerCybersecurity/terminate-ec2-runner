# Terminate EC2 Runner
Terminate EC2 instance that was used as GitHub self-hosted runner

This action works in conjunction with [provision-ec2-runner](https://github.com/gingercybersecurity/provision-ec2-runner) for complete lifecycle management.

## Getting Started

See the [README for provision-ec2-runner](https://github.com/GingerCybersecurity/provision-ec2-runner/blob/main/README.md) for the full description of prerequisites.

For examples of how to use `terminate-ec2-runner` in your workflow, see example workflow configurations [here](https://github.com/GingerCybersecurity/provision-ec2-runner/tree/main/examples).
